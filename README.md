# _printf

_printf is a function that produces output according to a format.


## Installation
### _printf
Copy files to you computer and call the functions.
### man page
sudo cp _printf.1 /usr/share/man/man1/
sudo mandb 

## Usage

``` C
 ui = 2147484671
 _printf("Unsigned:[%u]\n", ui);

# returns 'Unsigned:[2147484671]'

```
